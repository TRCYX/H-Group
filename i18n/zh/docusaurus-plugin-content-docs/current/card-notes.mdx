本页是 [Hanab Live 文档对笔记功能的描述](https://github.com/Hanabi-Live/hanabi-live/blob/main/docs/features.md#notes)的中文翻译。

- 注意：文中提到标点符号时，都应视作半角。

## 基本描述

- 游戏中，玩家可以右击或双击任何一张牌以添加笔记。之后将鼠标悬停在牌上时，会显示一个包含笔记内容的提示框。
- 这适用于记录牌的场合信息作为备忘。（例如，“他应该知道这张牌是红 2，因为这是第 3 回合的提示”）
- 使用游戏内回放时也可以写笔记，可以以此追踪某张牌在手中的移动情况。
- 笔记由服务器记录，玩家可以在游戏中途切换设备而保留所有已写的笔记。
- 笔记会存入数据库，在复盘时留存。
- 所有人的笔记会一并显示给旁观者，看起来会很有趣。

## 完全信息笔记

- 如果笔记匹配一张牌的英文名称（例如 `red 1`、`r1` 等），牌面会相应变化。
  - 如果收到的提示与笔记内容不符，牌面会自动恢复原状。
- 如果不希望出现这种行为，可以在笔记末尾添加问号。（例如 `r1?`）

- 如果有两个花色使用相同的缩写，第二个花色的缩写将改为最左侧未使用的字母。
  - 例如，在“彩色 & 混浊彩色（6 花色）”中，泥泞彩色（Muddy Rainbow）的缩写将变为“U”。
- 译注：进入游戏后，可以将鼠标悬停在打牌堆上，看到每个花色对应的缩写。

## 部分信息笔记

- 如果你无法明确一张牌的信息，但已经将其范围缩小到几种可能性，也可以在牌上以笔记形式表达。
- 列出所有可能性，用逗号分隔。
- 例如，在一张红牌上写 `r2, r3` 会告诉游戏“尽管这张牌只有红色提示，但将其展示为带有对 1、4、5 的否定信息”。
  - 译注：也可以写 `r23`、`gr4`、`by35` 等
- 你也可以用感叹号排除某些可能性。
- 例如，笔记 `!r2` 会排除这张牌是红 2 的可能性。

## 特殊笔记边框

- 笔记 `f` 表示这张牌被*飞*，这会在牌周围绘制一个特殊边框。
- 笔记 `cm` 表示这张牌*已被弃牌位移过*，这会在牌周围绘制一个特殊边框。

## 其他特殊笔记

- 笔记 `kt`、`trash`、`stale` 或 `bad` 表示这张牌是“废牌”，这会在牌上绘制一个特殊图标。
- 笔记 `fix`、`fixme` 或 `needs fix` 表示这张牌需要在未来某个时刻*修正*，这会在牌上绘制一个特殊图标。
- 笔记 `?` 表示这张牌上有不确定的信息，这会在牌上绘制一个特殊图标。
- 笔记 `!` 表示这张牌上有确定的信息，这会在牌上绘制一个特殊图标。
- 笔记 `blank` 可以让牌看起来像牌堆背面。
- 笔记 `unclued` 或 `x` 可以手动移除牌被提示后通常显示的边框。
- 笔记 `clued` 可以手动提升牌并添加牌被提示后通常显示的边框。

## 为特殊笔记添加上下文

- 如果你想保留特殊笔记的行为，但同时在笔记中添加额外文本，可以将特殊笔记放在方括号中。（例如 `[r2] 第 3 回合已知`）
- 此功能还可用于叠加多个效果。（例如 `[r1] [f]` 会将牌标记为红 1 并添加*飞牌*边框）

## 笔记快捷键

还有一些用于创建笔记的键盘快捷键：

- `Shift + 右键单击` → `[f]`
- `Alt + 右键单击` → `[cm]`
- `Ctrl + Alt + 右键单击` → 插入回合计数
- `Ctrl + Shift + 右键单击` → 重复上次输入的笔记
  - 如果需要在多张牌上输入相同的笔记，可以在第一张牌上输入笔记，然后在其他牌上使用此快捷键。
- `Ctrl + Shift + Alt + 右键单击` → `[kt]`
