## 特殊行动 \{#special-moves\}

### 有价值的误导性提示 \{#the-duplicitous-value-clue\}

- 通常我们不会对重复的牌给出*出牌提示*，因为这会违反*良触原则*。
- 但在某些情况下，有可能向 Bob 给出*出牌提示*，诱导他打出一张重复的牌。如果这个提示同时还触及其他有价值的牌，那么这就会是一次*有价值的误导性提示*。
- 我们约定只要这种提示满足*提示价值下限原则*（不计算重复的牌本身），就不会引发后续的*引牌*或盲打——这个提示的目的只是“锁定”其他有价值的牌。
- 例如在 3 人游戏中：
  - 红 2 和蓝 2 已经打出。
  - Alice 持有一张全员已知的红 3。
  - Alice 提示 Bob 数字 3 作为*出牌提示*，提示到 2 张牌。
  - Bob 打出焦点处的 3，以为它是蓝 3。但它实际上是红 3，且成功打出。
  - 虽然这出乎意料，但大家明白这是一个*有价值的误导性提示*，因为“锁定”Bob 的另一张 3 价值足够。
  - Bob 将另一张 3 标记为**任意**有用的 3。它并不保证可打出。（从而没有人会认为这是*引牌*或*飞牌*。）

- 对于**不**满足*提示价值下限原则*的*误导性提示*，请参阅下面的[_误导性节奏提示_](#the-duplicitous-tempo-clue)部分。
  - 译注：原文写的是不满足*提示价值下限原则*的 _Duplicitous Value Clue_，即*有价值的误导性提示*。译者认为下面也未引入就使用了 _Duplicitous Clue_，且一开始的 [Google 文档](https://docs.google.com/document/d/1zhFHjb2ChoLEIKn71bd8yKFczy0kn3hStrydRQ_m4NE) 中同一位置使用的也是 _Duplicitous Clue_，没有价值的 _Duplicitous Value Clue_ 听起来很奇怪，所以其实这类提示可能应统称 _Duplicitous Clue_ 而非 _Duplicitous Value Clue_，译作*误导性提示*。

### 误导性盲打 \{#the-duplicitous-blind-play\}

- 首先，请参阅[_有价值的误导性提示_](#the-duplicitous-value-clue)的章节。
- *有价值的误导性提示*直接提示到一张重复的牌（为了“获取”其他牌）。类似地，也可以让一张重复的牌盲打（为了“获取”其他牌）。这被称为*误导性盲打*。
- *误导性盲打*与*有价值的误导性提示*的运作方式相同——只要提示满足*提示价值下限原则*，就不会引发后续的*引牌*或盲打。
- 例如在 3 人游戏中：
  - 红 2 和蓝 2 已经打出。
  - Alice 持有一张全员已知的红 3。
  - Alice 提示 Cathy 蓝色，提示到一张蓝 4，作为*出牌提示*。
  - Bob 盲打*飞牌位*，以为是*飞*蓝 3。结果却是一张红 3，且成功打出。
  - 虽然这出乎意料，但大家明白这是一次*误导性盲打*，因为“锁定”Cathy 的蓝 4 价值足够。
  - 大家都知道蓝 4 并不保证可打出。（从而没人会认为这是*引牌*或*飞牌*。）

<DuplicitousBlindPlay />

- 注意*误导性盲打*也可以通过某些*饵牌*、*钓牌*等执行。

### 误导性节奏提示 \{#the-duplicitous-tempo-clue\}

- 首先，请参阅[_有价值的误导性提示_](#the-duplicitous-value-clue)和[_误导性盲打_](#the-duplicitous-blind-play)的章节。
- 在*有价值的误导性提示*和*误导性盲打*中，提示会触及至少一张“额外”的有价值的牌。但如果这类行动执行时，没有触及额外的有价值的牌，会怎样呢？
- 如果一张已提示过的牌被填补了信息，揭示为可打出，那么这个提示应被追溯视为*节奏提示*。这类提示称为*误导性节奏提示*。
- *误导性节奏提示*可能会也可能不会引发*节奏提示弃牌位移动*。（判断条件与“普通”*节奏提示*相同。）
- 例如在 3 人游戏中：
  - 红 1、绿 1 和蓝 1 已经打出。
  - Cathy 持有一张全员已知的红 2。
  - Bob 持有一张之前提示过的绿 2，但尚不知道其数字。
  - Alice 提示 Bob 数字 2，触及一张新牌，同时“填补”了绿 2 的信息。
  - Bob 知道这是对蓝 2 的*出牌提示*。他将其打出，但实际上是红 2，且成功打出。
  - Bob 的第一反应是这是*有价值的误导性提示*，但提示没有触及其他新牌。于是，Bob 明白这是*误导性节奏提示*。
  - 由于 Alice 没有处于有效的停顿状态，且 Bob 的绿 2 上得到的节奏并不算*有价值*，Bob 将此视为*节奏提示弃牌位移动*。（对绿 2 的*节奏提示*不算*有价值*，因为绿 2 可被*引牌*，且提示只获得一个单位的*节奏*。）

### 受助废牌弃牌位移动 \{#the-assisted-trash-chop-move\}

- 首先，请参阅[_废牌弃牌位移动_](level-4.mdx#the-trash-chop-move-tcm)的章节。
- *废牌弃牌位移动*可以非常有用，但有时无法干净地提示已知废牌，执行受阻。
- 这种情况下，可以通过使用*废牌诈飞*或*未知废牌钓牌*盲打一张重复牌来解决问题。这被称为*受助废牌弃牌位移动*。
- 由于提示没有触及“额外”的牌（如在*有价值的误导性提示*中那样），大家应该清楚这个提示的意图：对被提示的牌右侧的所有牌执行*废牌弃牌位移动*。
- 例如这是一局 3 人游戏中通过*废牌飞牌*执行的*受助废牌弃牌位移动*：
  - 蓝 1 已经打出。
  - Alice 持有一张全员已知的黄 1。
  - Alice 提示 Cathy 数字 1，提示到 3 号位的蓝 1，作为*出牌提示*。
  - Bob 知道既然蓝 1 被展示为一张好 1，这一定是在*废牌飞牌*，且 Bob 的*飞牌位*上必须是红 1、绿 1、黄 1 或紫 1。
  - Bob 盲打*飞牌位*。出乎他的意料，这是一张黄 1，且成功打出。
  - 由于黄 1 重复了，其他人知道发生了不寻常的事情。但由于提示没有触及额外的牌，所有人都明白这是对 Cathy 的*受助弃牌位移动*。（此场景中，Cathy 应该将弃牌位*移*过 4 号位和 5 号位。）

- 例如这是一局 3 人游戏中通过*未知废牌钓牌*执行的*受助弃牌位移动*：
  - 蓝 1 已经打出。
  - Alice 持有一张全员已知的黄 1。
  - Alice 提示 Cathy 蓝色，提示到 2 号位的蓝 1，作为*出牌提示*。
  - Bob 知道既然蓝 1 被表示为蓝 2，这一定是在*未知废牌释放*。
  - Bob 盲打*第三飞牌位*。出乎他的意料，这是一张黄 1，且成功打出。
  - 由于黄 1 重复了，其他人知道发生了不寻常的事情。但由于提示没有触及额外的牌，所有人都明白这是对 Cathy 的*受助弃牌位移动*。（此场景中，Cathy 应该将弃牌位*移*过 3 号位、4 号位和 5 号位。）

### 时间旅行弃牌位移动（直接形式） \{#the-time-travel-chop-move-direct-form\}

- 首先，请参阅[_有价值的误导性提示_](#the-duplicitous-value-clue)和[_误导性盲打_](#the-duplicitous-blind-play)的章节。
- 通常*误导性提示*会“获取”额外牌或作为*节奏提示*填补另一张牌的信息。然而，如果执行的*误导性提示*没有达成其中任意一者，那一定是出于其他原因。
- 这种情况下，在重复的牌展示出来之后，被提示的玩家应将此提示视为对自己手牌的*废牌弃牌位移动*。（换句话说，应该假装重复的牌是被直接以*废牌弃牌位移动*提示。）
- 例如在 3 人游戏中：
  - 所有的 2 都已经打出。
  - Cathy 持有一张全员已知的红 3。
  - Alice 提示 Bob 数字 3，提示到一张新 3，作为*出牌提示*。
  - Bob 打出他的 3，期望它是红 3 以外的任何 3。但实际上是红 3，且成功打出。
  - 由于这次行动是 0 换 1，也没有在任何一张牌上提供节奏，所有人都明白这是*时间旅行弃牌位移动*。Bob 将弃牌位*移*过自己红 3 后面的所有牌。

### 时间旅行弃牌位移动（盲打形式） \{#the-time-travel-chop-move-blind-play-form\}

- 首先，请参阅[_时间旅行弃牌位移动（直接形式）_](#the-time-travel-chop-move-direct-form)的章节。
- 通常*时间旅行弃牌位移动*会直接提示重复的牌。但如果重复的牌受阻，无法干净地提示呢？
- 这种情况下，玩家可以让重复的牌盲打而非直接提示，从而达到相同效果（即弃牌位*移*过重复牌右侧所有牌）。然而，需要注意确保最终结果真的足够“坏”，使得不被误解为其他类型的提示。
- 具体来说：
  - 提示必须不满足*提示价值下限原则*（类似于*误导性提示*）。
    - 译注：自然也不能像*误导性节奏提示*。
  - 提示必须看起来不像对接收提示者的[_受助废牌弃牌位移动_](#the-assisted-trash-chop-move)。
- 例如在 3 人游戏中：
  - 所有的 1 都已经打出。
  - Cathy 有一张已知的蓝 2。
  - Bob 手中有一张之前提示过的红牌，但不知道其数字。
  - Alice 提示 Bob 数字 3，“填补”红牌为红 3。（提示没有触及其他牌。）
  - Bob 知道这是对红 3 *飞*红 2。Bob 盲打*飞牌位*，但结果是蓝 2 而非红 2，且成功打出。
  - 由于这次行动是 0 换 1，Bob 明白自己应该将弃牌位*移*过蓝 2 后面的所有牌，作为*时间旅行弃牌位移动*。
