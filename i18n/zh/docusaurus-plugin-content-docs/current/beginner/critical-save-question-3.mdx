<TabItem value="question">

- Alice 提示 Bob 黄色，仅提示到他 4 号位的牌。
- 这是**出牌提示**还是**留牌提示**？
- Bob 会写下什么**笔记**？（请具体说明。）

</TabItem>
<TabItem value="solution">

- Bob 知道这可能是一个对黄 1 的**出牌提示**。
- 然而，提示触及了他的弃牌位，这也可能是一个**留牌提示**。由于黄 3 和黄 4 在弃牌堆中，提示的黄色能够与它们匹配，因此也可能是黄 3 或黄 4。
- Bob 随后注意到 Cathy 手中有一张黄 3。由于只有两张黄 3，Bob 不可能再有黄 3。
- Bob 不知道这是**出牌提示**还是**留牌提示**，但他必须暂时将其视为**留牌提示**，直到获得更多信息。
- Bob 会写下包含以下可能性的**笔记**：
  - 黄 1（作为**出牌提示**）
  - 黄 4（作为**留关键牌提示**）

</TabItem>
