<TabItem value="question">

- Alice 提示 Bob 数字 2，仅触及他的 4 号位。
- 从 Bob 的角度来看，这是**出牌提示**还是**留牌提示**？
- Bob 的**笔记**会是什么内容？

</TabItem>
<TabItem value="solution">

- 这不可能是**留牌提示**，因为提示没有触及 Bob 的弃牌位。因此一定是**出牌提示**。
- Bob 的**笔记**会包含以下可能性：
  - 蓝 2（可直接打出）
  - 黄 2（作为通过 Cathy 的黄 1 给出的**延迟出牌提示**）

</TabItem>
