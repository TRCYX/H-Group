<TabItem value="question">

- Alice 应该采取什么行动？

</TabItem>
<TabItem value="solution">

- Alice 需要在 Bob 弃掉蓝 3 之前给出一个**出牌提示**。
- 如果 Alice 提示蓝色，提示会聚焦在蓝 3，但蓝 2 会重复（因为它已经打出）。
- 如果 Alice 提示数字 3，提示会聚焦在蓝 3，但也会触及红 3。红 3 未来会有用，因此这是好事。
- 因此，Alice 应该提示 Bob 数字 3。

</TabItem>
