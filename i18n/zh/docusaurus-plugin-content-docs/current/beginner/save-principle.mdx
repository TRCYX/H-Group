- 之前我们解释过，团队必须通过*留牌提示*来保留某些类型的牌。
- 但除此之外，我们还约定必须获取所有可打出的牌。换句话说，<Em>**我们不会弃掉可打出的牌！**</Em>
- 因此，我们<strong><Em>绝对</Em>不能弃掉</strong>的牌的完整列表如下：
  - 所有 5
  - 唯一的 2（即当前只有一张可见的 2）
  - 关键牌（即弃到只剩一张的牌）
  - 唯一的可打出的牌（即当前只有一张可见的可打出的牌）
- 不让对方弃掉重要的牌被称为*留牌原则（Save Principle）*。<Em>**这是所有原则中最重要的**</Em>。你可以**打破任何其他原则或规定**来避免列表中的牌被弃掉。事实上，有时你只能违反*良触原则*来保留特定牌，这是可以接受的。

### 违反良触原则 \{#violating-good-touch-principle\}

- 在下面的例子里：
  - Alice 需要保留 Bob 弃牌位上的最后一张蓝 3。
  - Alice 知道她可以通过颜色提示或数字提示来执行*留关键牌提示*。
  - 如果 Alice 提示蓝色，它会违反*良触原则*，因为它会提示到两张蓝 4。
  - 如果 Alice 提示 3，它也会违反*良触原则*，因为它会提示到红 3（而红 3 已经在牌堆上打出）。
  - 在这里，Alice 知道违反*良触原则*比失去得到满分的机会要好。因此，Alice 必须在两害相权中做出抉择，决定是提示蓝色还是 3。
  - Alice 决定提示 Bob 蓝色而不是 3，因为至少其中一张蓝 4 之后会有用。
