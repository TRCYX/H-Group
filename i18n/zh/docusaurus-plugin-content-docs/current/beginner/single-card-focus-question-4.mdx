- Bob 的手中有一张提示过的 5 和一张提示过的 2。
- Alice 提示 Bob 红色，提示到三张牌。
- 哪一张牌是焦点？

</TabItem>
<TabItem value="solution">

- 3 号位，因为有两张新牌，且 3 号位是弃牌位。
  - （5 号位之前提示过，因此红色提示仅仅填补了信息。）

</TabItem>
