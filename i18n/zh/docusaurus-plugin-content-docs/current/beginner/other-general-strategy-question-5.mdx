<TabItem value="question">

- 列出 Alice 可以给出的所有合法提示。
- Alice 应该从中选择哪个提示？

</TabItem>
<TabItem value="solution">

- Alice 可以提示 Bob 绿色，作为对绿 3 的**出牌提示**。
  - 这会覆盖一张牌。
- Alice 可以提示 Bob 数字 3，作为对绿 3 的**出牌提示**。
  - 这会覆盖两张牌。（Bob 会打出绿 3 并保留黄 3。）
- Alice 知道通常颜色提示优先于数字提示。但这只在颜色提示和数字提示大致相同时才适用。在这里，覆盖两张牌远好于覆盖一张牌，所以 Alice 应该提示数字 3。

</TabItem>
