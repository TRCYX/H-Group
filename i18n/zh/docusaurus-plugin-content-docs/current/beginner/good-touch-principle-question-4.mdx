<TabItem value="question">

- 需要打出绿 3。
- Alice 应该采取什么行动？
- 如果 Alice 应该提示，提示的焦点上应该写下什么**笔记**？

</TabItem>
<TabItem value="solution">

- 无论是颜色提示还是数字提示，都会使绿 3 重复，违反**良触原则**。
- 因此，Alice 应该耐心等待 Bob 弃掉绿 3。一旦第一张绿 3 被弃掉，大家就可以提示剩下的绿 3。
- 由于无法给出允许的提示，Alice 必须弃牌。

</TabItem>
