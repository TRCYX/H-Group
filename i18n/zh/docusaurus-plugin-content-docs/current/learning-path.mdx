### 介绍 \{#intro\}

世界各地都有喜欢玩花火的人。得益于互联网，我们经常看到各式各样的新玩家。有人只玩几天，也有人持续玩很多年！但无论玩多久，总有新东西可以学习，因为花火像象棋一样具有深度。

对于热爱花火并希望继续与我们的团体一起玩的人来说，从新手到专家的阶梯很漫长。它充满许多练习、无数的错误以及许多不同的新约定。可能听起来有点吓人，但这并不是本意——无论你水平高低，认识不同的人、玩很多花火总是很有趣的。

与新玩家一起玩时，团体中的每个人都知道要**保持简单**，只使用[入门指南](beginner.mdx)涵盖的基本约定。新人玩过几局初学者游戏后，事情就会变得“开放”一点，开始教授新的特殊行动，比如*弃牌位移动*和*多层飞牌*。有很多东西要学，我们不想一次性让新玩家感到不知所措。那么，学习的正确顺序是什么？

方便的是，我们根据经验将约定分为不同的“等级”。游戏开始之前，新玩家<Em>**应该向队伍明确说明想要玩的等级**</Em>。然后队伍中的每个人都应该<Em>**坚持只使用该等级以下的约定**</Em>。对于高级玩家来说，向新玩家展示高级技巧可能很诱人。但不要这样做！保持简单！让他们先学习基础，之后再引入更复杂的内容。

为什么这很重要？我们从中级玩家那里看到的一个常见错误是，一旦学到了高级技巧，他们就认为自己已经理解了其原理，然后在完全不合适的场合使用。正如俗话所说，“先学走，再学跑”。与只使用最基本的约定、做出可预测的举动并缓步融入新内容的中级玩家相比，和那些知道了高级技巧但规划能力和游戏理解较弱的中级玩家一起玩体验**远远更糟**。

如果你是新玩家，觉得自己已经牢牢掌握了入门指南中的概念，请使用左侧边栏，开始探索等&NoBreak;级&nbsp;2 的约定。

有关各变种如何与等级系统相关联，请参阅[变种总结](#variant-summary)。

### 变种概要 \{#variant-summary\}

[Hanab Live 有许多不同的变种](https://github.com/Hanabi-Live/hanabi-live/blob/main/docs/variants.md)，H-Group 会混合着打这些变种。变种的目的是增加游戏难度，为老玩家提供额外挑战。就像在尚未掌握基础机制之前使用高级约定会适得其反一样，玩复杂的变种同样会适得其反。因此，初学者应尽可能坚持玩原版（No Variant），甚至可以一直玩到最高等级。

话虽如此，我们了解玩家通常在达到最高等级之前就会开始厌倦原版。（他们也会被其他厌倦原版的老玩家鼓动去玩变种。）较低等级的玩家可以偶尔混着玩一玩带有彩色、白色和黑色的变种。但大多数时候，应该坚持玩原版。

以下是一些变种如何与约定等级对应的大致描述。

| 变种       | 等级 |
| ---------- | ---- |
| 原版       | 1    |
| 3/4/6 花色 | 2    |
| 彩色       | 4    |
| 白色       | 4    |
| 黑色       | 5    |
| 粉色       | 16   |
| 棕色       | 16   |
| Omni       | 16   |
| Null       | 最高 |
