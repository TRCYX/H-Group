[_毒瘤双重诈飞_](../level-15.mdx#the-pestilent-double-bluff-pdb)是一个等级 15 的约定。

### 例 1 \{#example-1\}

![示例截图](/img/examples/pestilent-double-bluff.png)

- 现在是游戏的第一回合，{"Dr_Kakashi"} 先行动。
- 试想 {"Dr_Kakashi"} 提示 RaKXeR 数字 2。这会提示到红 2，作为*出牌提示*。
- 下一名玩家是 MasN。从 MasN 的视角来看，这看起来像是在*飞牌*。MasN 知道自己*飞牌位*上可能有一张红 1。MasN 盲打*飞牌位*，结果是黄 1，成功打出。MasN 现在知道自己被*诈飞*了。
- 下一名玩家是 Kyle。Kyle 看到，从 RaKXeR 的视角来看，刚刚发生了一次*飞牌*，而 RaKXeR 手中的 2 是黄 2（与刚刚盲打的黄 1 相连）。这是一个问题（因为这张牌实际上是红 2，RaKXeR 将会误打）。
- Kyle 知道 {"Dr_Kakashi"} 一定是在试图传达一些额外的信息。这种情况发生时，通常是*良触诈飞*，意味着其他人手中已经有一张黄 2 提示过了。然而并非如此，因为 {"Dr_Kakashi"} 是在游戏的第一回合给出提示的，显然此时还没有人手里有牌提示过。
- 这一定是*毒瘤双重诈飞*。Kyle 可以盲打*飞牌位*来“治好”这个局面，而这正是 {"Dr_Kakashi"} 的意图。Kyle 盲打*飞牌位*，结果是蓝 1，成功打出。
- 从 RaKXeR 的视角来看，{"Dr_Kakashi"} 的行动在 MasN 盲打黄 1 后看起来像是*飞牌*。然而，在 Kyle 也盲打一张牌后，RaKXeR 惊讶地倒吸一口凉气。一定出现了特殊情况，因为从黄 1 到黄 2 的*飞牌*不可能会让 Kyle 盲打。
- 因此，RaKXeR 知道自己手中的 2 不可能是黄 2。进一步，RaKXeR 还知道这张 2 也不可能是蓝 2，因为不然 MasN 就不会盲打。
- 于是，RaKXeR 知道这一定是*毒瘤双重诈飞*，将自己的 2 标记为绿 2、红 2 或紫 2。

### 例 2 \{#example-2\}

![示例截图](/img/examples/pestilent-double-bluff-2.png)

- 现在是 {"Dr_Kakashi"} 的回合。
- 试想 {"Dr_Kakashi"} 提示 Kyle 数字 3。这会触及蓝 3 作为*出牌提示*。
- 下一名玩家是 Sucubis。他知道这是在*飞牌*，因此他盲打*飞牌位*（1 号位）作为蓝 1。结果却是一张黑 2，并且成功打出。Sucubis 现在知道自己被*诈飞*了。
- 下一名玩家是 Valetta6789。她知道刚刚出现了奇怪的事情。由于黑 2 与数字 3 提示相连，Kyle 会认为自己持有黑 3（并将会误打）。
- Valetta6789 知道这一定是在*毒瘤双重诈飞*。她盲打*飞牌位*。是紫 1，成功打出。
- 下一名玩家是 Kyle。在数字 3 提示给出而黑 2 盲打后，Kyle 觉得自己手中的 3 是黑 3（因为 2 与 3 相连）。然而，如果它是黑 3，Valetta6789 就不会盲打。因此，Kyle 知道自己手中的 3 一定是除黑 3 外其他颜色的 3，他将其如是标记。

### 例 3 \{#example-3\}

![示例截图](/img/examples/pestilent-double-bluff-3.png)

- 现在是 scharkbite 的回合。
- 试想 scharkbite 提示 asaelr 数字 3。这会提示到黑 3，作为*出牌提示*。
- 下一名玩家是 iamwhoiamhahaha。他知道这是在*飞牌*，因此他盲打*飞牌位*（3 号位）作为黑 2。结果却是一张红 2，成功打出。iamwhoiamhahaha 现在知道自己被*诈飞*了。
- 下一名玩家是 Romain672。他知道刚刚出现了奇怪的事情。由于红 2 与数字 3 提示相连，asaelr 会认为自己持有红 3（将会误打）。
- 然而，Romain672 知道 asaelr **不会**认为自己有红 3，因为 scharkbite 自己手中有两张红 3。
- 尽管如此，scharkbite 并不知道自己手中有两张红 3；从 scharkbite 的视角来看，asaelr **会**认为自己有红 3。
- Romain672 知道 scharkbite 绝不会故意给出如此糟糕的提示——他一定在试图传达一些额外的信息。
- Romain672 知道这一定是在*毒瘤双重诈飞*。他盲打*飞牌位*（1 号位）。是紫 2，成功打出。
- 下一名玩家是 asaelr。在数字 3 提示给出且红 2 被盲打后，asaelr 觉得自己手中的 3 是红 3（因为 2 与 3 相连）。然而，如果这是红 3，那么 Romain672 就不会盲打。因此，asaelr 知道自己手中的 3 一定是除红 3 外其他颜色的 3，他将其如是标记。
